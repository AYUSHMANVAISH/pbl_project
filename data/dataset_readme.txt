# Dataset Information

## Important Notice

The dataset used in this project is very large and cannot be uploaded to GitHub because of file size limitations. For this reason, the dataset files are not included in this repository.

---

## Dataset Source

The dataset can be downloaded from the official source at the link below:

CICIDS 2017 Dataset  
http://205.174.165.80/CICDataset/CIC-IDS-2017/Dataset/

---

## Usage Instructions

1. Download the dataset from the link above.
2. Extract the downloaded files.
3. Place the CSV files in the appropriate folder before running the project.

---

## Note

Please make sure you have enough storage space and a stable internet connection before downloading, as the dataset size is large. The dataset is intended for academic and research purposes only.
